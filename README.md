# neuron-web-simulation
A simulation of biological neuron. Allows visualization of basic neuron mechanics.

## Project status (big picture)
The final project goal is a full neuron simulation that combines interactive visualization with simplified physics-based behavior, including membrane dynamics, signal propagation, and particle-like activity that makes the model feel alive and educational rather than static. In the completed version, users should be able to observe and tweak core parameters, then see how those changes affect neuron behavior in real time.

Right now, development is in the foundational stage: core rendering, particle systems, UI scaffolding, and project structure are in place, and the focus is on building reliable simulation primitives before expanding into more complete biological/physical neuron mechanics. The current milestone is establishing a stable base so future iterations can add richer physics, better accuracy, and clearer controls without major rewrites.
